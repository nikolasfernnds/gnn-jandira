# GNN Jandira: Módulo de Ocorrências Urbanas

## Visão Geral do Projeto

O **GNN Jandira** é um sistema web projetado para ser um canal centralizado de comunicação entre os cidadãos e a administração pública de Jandira. Seu principal objetivo é permitir que os cidadãos registrem, reportem e acompanhem o status de problemas de infraestrutura e manutenção urbana.

O sistema visa facilitar a comunicação entre os cidadãos e a administração pública, atuando como um canal centralizado para o registro de ocorrências. Isso contribuirá para a resolução mais rápida e eficaz dos problemas urbanos, promovendo maior engajamento cívico.
